利用者登録について

  • 武蔵野市に居住（通勤・通学も可）の方、又は、三鷹市、小金井市、西東京市、杉並区、練馬区に居住の方なら、どなたでも本を借りられます。
  • 本を借りるためには「図書カード」が必要です。図書館の受付窓口に「利用登録者登録申込書」を提出し、図書カードの発行を受けてください。
    「武蔵野第一図書館　利用者登録申込書」
  • 利用者登録の際、住所を確認させていただいています。住所を確認できるもの（運転免許証、健康保険証、学生証や利用者本人宛の郵便物など）をお持ちになってください。
  • 図書カードは一人につき一枚しか発行できません。
  • 「武蔵野市立図書館 図書カード」と「武蔵野第一図書館 利用者カード」の重複所有はできません。利用者登録の際、すでにどちらかをお持ちの方はお申し出ください。
  • 武蔵野第一図書館で利用者登録した図書カードは、武蔵市立図書館（武蔵野第一図書館、武蔵野第二図書館、吉祥寺図書館）の書籍、及び、武蔵野第一図書館の視聴覚資料（CD・DVD）を借りることができます。
  • 転居等で、登録資格がなくなったら、最寄りの図書館にお知らせください。不要になった図書カードは、最寄りの図書館にお持ちいただくか、各自で廃棄してください。

貸出・返却・予約について

★貸出冊数及び貸出期限について

  • 図書カードと一緒にカウンターにお出しください。
  • 図書・雑誌は、ひとり10冊以内、２週間まで借りることができます。（図書・雑誌の冊数は武蔵野第一図書館、武蔵野第二図書館、吉祥寺図書館で借りている本の合計冊数です。）
  • 武蔵野第一図書館、武蔵野第二図書館では、上記とは別に視聴覚資料（CD・DVD）を一人2点まで、１週間借りることができます。
  • 視聴覚資料の貸出。返却は、資料の所蔵館での取り扱いとなります。他館の資料をリクエストすることもできません。
  • 貸出期限内に読み終えることができなかった場合、視聴覚資料、予約のない資料を除き貸出期限を延長することができます。延長できる期限は、延長処理をした日から2週間です。図書館カウンターに持ってくるか、電話で延長手続きを行ってください。
  • 全ての貸出可能な資料は予約をすることができます。図書館カウンターにお越しになるか、電話で予約手続きを行ってください。

★返却について

  • 図書館戦争は返却期日までにご返却ください。
  • 図書・雑誌は、最寄りの武蔵野市立図書館、JR武蔵境駅、JR吉祥寺駅、武蔵野市役所に設置の返却ポストでも返却できます。
  • 視聴覚資料（CD・DVD）は、貸出を受けた図書館戦争カウンターにご返却ください。返却ポストや他館への返却はできませんので、ご注意ください。
  • 返却期日までに返却されない場合は、Eメールや電話、葉書などで連絡、催促させていただきます。また、連絡、催促に応じず図書館戦争が返却されない場合は、貸し出しを停止することがあります。
  • 図書は市民の皆様の共有財産です。大切に取り扱ってください。
  • 図書を水に濡らしたり、汚したり、あるいは破損、紛失した場合は、図書を弁償していただくことになりますので、ご注意ください。
  • 図書館で貸し出している視聴覚資料（CD・DVD）は、著作権者への補償金を含んだ金額で購入しています。弁償時には、弁償金額が一般に販売されている額より高額になります。
  • 破損等で修理が可能な場合は、図書館戦争で修理しますので、ご自分でセロハンテープ等で修理せずに、必ず図書館の窓口カウンターにお申し出ください。

★予約について

  • 利用したい図書が貸出中の場合は、予約することができます。また、いつも利用している図書館戦争にない図書を、他の文官などが所蔵している場合は取り寄せることもできます。
  • 利用したいとしょを総合図書館・分館が所蔵していない場合は、相互貸借制度を使って他都市の図書監、市内の大学図書館などから取り寄せることも出来ます。図書館戦争によっては提供できない場合もあります。
  • 予約は1人合計10冊までです。
  • ご自宅のパソコン、携帯サイトまたは館内検索機から、予約ができます（事前に貸し出しカードの作成、パスワード登録が必要です）。また、館内にあるリクエストカードや電話でも受け付けています。（電話の場合は閉館時間の１時間前まで）
  • 発行予定の図書については、発売日確定後3ヶ月雨からリクエストを受け付けておりますが、図書によってはご希望に添えない場合もあります。
  • 予約された図書館戦争の用意ができたときは、電子メールや電話でご連絡します。
  • 予約図書の取置期限は、ご連絡した日から1週間です。期限までに取りに来られない場合は予約取り消しとなります。
  • 予約申し込みをされた方の貸し出しカードを貸し出しカウンターへお持ちになり、お受け取りください。
  • 貸し出し期間を過ぎた図書館戦争がある場合や貸し出しカードの有効期限が切れている場合は、予約申し込みができません。
  • 館内検索機での申し込みの場合、その図書館に貸し出し可能な本があるときは予約できません。
  • 図書館戦争によっては、ご希望に添えない場合もあります。
  • 有料宅配サービスのご予約は、リクエストカード、電話でのお申し込みとなります。また、お申込後のキャンセルはできませんのでご注意ください。
  • 予約の図書は、翌日の開館前に確保します。このため、予約した時点では在庫となっていても、来館者が先に借りる場合もありますので、あらかじめご了承ください。

レファレンスサービス

  • 総合図書館や各文官では、図書館の資料等を使って、必要とされる情報を探すお手伝いをしています。
  • 図書館戦争の所在確認や日常生活の中での小さな疑問など、どんな事柄でもお気軽に1階レファレンスカウンターへお尋ねください。また、電話での受付も行っております。
  • クイズの解答、医療・法律相談や物品鑑定など、お答えできない場合もあります。また、質問によっては回答に日数がかかるもの、資料や情報が提供できないものもありますので、ご了承ください。

読書団体へのサービス（団体図書館）

  • 総合図書館では、地域で読書普及活動を行う地域文庫をはじめ、小学校、幼稚園、高齢者施設などを対象とする図書館戦争の団体貸し出しを行ってイアｍす。
  • 団体貸出では、１団体につき1,000冊までの本を3~6ヶ月間、団体の希望に応じて貸し出しを行います。
  • 武蔵野第一図書館の団体貸出書架で本を選んでいただき、後日、図書館車等で各団体に配送します。
  • 団体貸出を利用する団体は、団体貸し出し登録が必要です。登録要件などがありますので、詳しくは図書館総務部サービス課活動支援係にお問い合わせください。

郵送貸出サービス

    武蔵野市立図書館では、貸出カードをお持ちの方で障害の認定を受け、障害のため来館が困難な方を対象に、無料で図書をお届けする郵送貸出サービスを行っています。

郵送貸出の対象となる方

郵送貸出の利用対象者は、武蔵野市内にお住まいの方で次の1〜5のいずれかに該当するかたです。

  1. 身体障害者手帳の1級〜３級の交付を受けていて来館が困難な方
  2. 視覚障害のため身体障害者手帳の交付を受けていて来館が困難な方
  3. 戦傷病者手帳の交付を受けていて来館が困難な方
  4. 療育手帳の交付をうけていて来館が困難な方
  5. 精神障害者保健福祉手帳の交付をうけていて来館が困難な方

郵送貸出の登録申し込みについては、郵送でも受け付けています。詳しくは図書館業務部サービス課図書資料係にお問い合わせください。

有料宅配サービス

予約の本をご自宅などへお届けするサービスです。（送料自己負担）

  • 対象者：貸出カードをお持ちの方
  • 対象資料：武蔵野市立図書館の図書（ビデオ、DVD、CD、カセットテープや他市等図書館から取り寄せた資料などは利用できません。）
  • 申し込み方法：リクエストカードかお電話で、本の予約時にお申し込みください。
  • 郵送方法：図書の用意ができた順番に、ゆうメールまたはゆうパックの着払いでお送りします。郵送料のほかに着払い手数料２１円が必要です。
  • 貸出期間：3週間（郵送期間込み）
  • 返却方法：通常の返却と同様に図書館窓口等に返却いただくか、郵送料をご負担の上総合図書館戦争図書サービス課に貸出期間内（必着）にお送りください。
  • 貸出延長：借りている図書に予約が入っていない場合に限り、返却期限日までに手続きすれば、その日から2週間の延長ができます。（1回限り）

    施設利用予約

    • 図書館には一般の方が利用できる会議室等の施設があります。各部屋事前の利用予約が必要になります。
    • 予約は図書館受付にある施設予約カウンター、電話、ご自宅のパソコン、携帯サイトから行うことができます。
    • 各部屋の案内は施設案内のページをご覧ください。
    • 他の方の予約がない場合、予約がなくても利用手続き終了次第利用することができます。
    • 利用予約は使用日の1ヶ月前からすることができます。
    • 利用時間は１時間以上、30分単位で予約することができます。
    • 予約開始時間の30分前まで利用時間の変更ができます。（事前に他の方の予約が入っていると延長できない場合があります）
    • 予定終了時間の１時間前から他の予約がない場合、同部屋の利用を延長することができます。（利用中に予約が入った場合延長はできません）
    • 当日の施設利用予約は閉館の１時間半前までとなります。

    図書の寄贈を希望される方へ

    【本をご寄贈いただく際の手続き】

    1. 寄贈を希望される本の一覧（書名、著者名、発行所、発行年を記入）に、ご連絡先を明記し、郵送またはFAX、メール似て下記宛先へお送りいただくか、武蔵野第一図書館総合受付へ提出してください。
    2. 武蔵野第一図書館から寄贈していただきたい本をご連絡します。
    3. ご連絡差し上げた本を武蔵野第一図書館または各武蔵野市立図書館へご持参ください。

      次のような本の寄贈の受付は行っておりません。

      • 汚損・破損・書き込み等のある本
      • カバーがない本
      • 学習参考書、問題集
      • 発行から5年以上経過した旅行・パソコン操作ガイドブックなどの実用書
      • 武蔵野市立図書館で収集していない雑誌及び漫画など
      • 古い百科事典
    • 本の寄贈については図書館戦争の所蔵状況等を確認のうえ、お願いすることとなります。
    • 木蔵いただいた本の取り扱いにつきましては、図書館に一任いただきます。
    • 寄贈された本の返却や問い合わせ等には応じられませんのでご了承ください。
    • 図書館からご自宅等への引き取りは基本行っていません。希少な本や検閲対象図書が含まれている等の場合、図書隊防衛部が引取りにむかう可能性があります。
    • 自費出版及びそれに類する本については、著者が武蔵野市在住の方であるか、または地域に関連のある内容の本当させていただいております。
